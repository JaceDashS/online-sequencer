# 단축키 케이스 테스트

## 범위
- 편집 및 모드 전환 단축키
- 실행 취소/다시 실행 단축키
- 선택 클립 삭제 단축키

## 환경
- 브라우저: Chromium 계열, Firefox
- OS: Windows
- 앱: Vite 개발 서버

## 테스트 데이터
- MIDI 파트가 있는 프로젝트
- 클립 선택이 가능한 상태

## 테스트 케이스

### SH-001 Quantize 토글 (Q)
사전조건: 이벤트 디스플레이가 활성화된 상태
절차:
1. Q 키를 누른다
2. Q 키를 다시 누른다
기대결과:
- Quantize 설정이 켜졌다가 꺼진다

### SH-002 Metronome 토글 (M)
사전조건: 이벤트 디스플레이가 활성화된 상태
절차:
1. M 키를 누른다
2. M 키를 다시 누른다
기대결과:
- Metronome 설정이 켜졌다가 꺼진다

### SH-003 Split 모드 토글 (3)
사전조건: MIDI 파트가 존재
절차:
1. 3 키를 누른다
2. 스플릿 프리뷰가 표시되는지 확인한다
3. 3 키를 다시 누른다
기대결과:
- Split 모드가 켜졌다가 꺼진다

### SH-004 Merge 모드 토글 (4)
사전조건: MIDI 파트가 존재
절차:
1. 4 키를 누른다
2. Merge 모드가 켜지는지 확인한다
3. 4 키를 다시 누른다
기대결과:
- Merge 모드가 켜졌다가 꺼진다

### SH-005 모드 종료 (1)
사전조건: Split 또는 Merge 모드가 켜져 있음
절차:
1. 1 키를 누른다
기대결과:
- Split/Merge 모드가 모두 꺼진다

### SH-006 선택 클립 삭제 (Delete/Backspace)
사전조건: 클립이 선택된 상태
절차:
1. Delete 또는 Backspace를 누른다
기대결과:
- 선택된 클립이 삭제된다
- 선택 상태가 해제된다

### SH-007 실행 취소 (Ctrl+Z / Cmd+Z)
사전조건: 클립 편집(분할/병합/이동/리사이즈) 수행
절차:
1. Ctrl+Z 또는 Cmd+Z를 누른다
기대결과:
- 마지막 편집이 되돌려진다

### SH-008 다시 실행 (Ctrl+Y / Ctrl+Shift+Z)
사전조건: 실행 취소를 1회 이상 수행
절차:
1. Ctrl+Y 또는 Ctrl+Shift+Z를 누른다
기대결과:
- 실행 취소된 편집이 복원된다
