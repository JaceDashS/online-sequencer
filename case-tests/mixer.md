# 믹서 케이스 테스트

## 범위
- 채널 볼륨/팬 조절
- 마스터 채널 조절
- 이펙트 체인/이펙트 토글
- 선택 트랙 동기화
- 믹서 패널 표시/리사이즈

## 환경
- 브라우저: Chromium 계열, Firefox
- OS: Windows
- 앱: Vite 개발 서버

## 테스트 데이터
- 트랙 2개 이상이 있는 프로젝트
- 믹서 패널이 보이는 상태

## 테스트 케이스

### MX-001 채널 볼륨 조절
사전조건: 믹서 패널 표시 상태
절차:
1. 트랙 1 채널의 볼륨을 올린다
2. 트랙 1 채널의 볼륨을 내린다
기대결과:
- 볼륨 값이 변경된다

### MX-002 채널 패닝 조절
사전조건: 믹서 패널 표시 상태
절차:
1. 트랙 1 채널의 패닝을 좌우로 변경한다
기대결과:
- 패닝 값이 변경된다

### MX-003 마스터 볼륨 조절
사전조건: 믹서 패널 표시 상태
절차:
1. 마스터 채널의 볼륨을 올린다
2. 마스터 채널의 볼륨을 내린다
기대결과:
- 마스터 볼륨 값이 변경된다

### MX-004 마스터 패닝 조절
사전조건: 믹서 패널 표시 상태

### MX-005 이펙트 체인 추가/삭제
사전조건: 믹서 패널 표시 상태
절차:
1. 트랙 1 채널에 이펙트를 추가한다
2. 추가한 이펙트를 삭제한다
기대결과:
- 이펙트가 추가되고 제거된다

### MX-006 이펙트 활성/비활성 토글
사전조건: 트랙에 이펙트가 1개 이상 존재
절차:
1. 이펙트를 비활성화한다
2. 이펙트를 다시 활성화한다
기대결과:
- 이펙트 활성 상태가 토글된다

### MX-007 선택 트랙 동기화
사전조건: 트랙 리스트와 믹서 패널이 동시에 표시
절차:
1. 트랙 리스트에서 트랙 1을 선택한다
2. 믹서에서 선택 상태가 반영되는지 확인한다
기대결과:
- 믹서 채널 선택 상태가 트랙 선택과 동기화된다

### MX-008 믹서 패널 표시 토글
사전조건: 툴바 표시 상태
절차:
1. 믹서 패널 토글을 꺼서 숨긴다
2. 다시 켜서 표시한다
기대결과:
- 믹서 패널이 정상적으로 숨김/표시된다

### MX-009 믹서 패널 리사이즈
사전조건: 믹서 패널 표시 상태
절차:
1. 믹서 리사이즈 핸들을 드래그해 높이를 늘린다
2. 다시 드래그해 높이를 줄인다
기대결과:
- 최소/최대 범위 내에서 높이가 부드럽게 변경된다

### MX-010 트랙 뮤트 (재생 중)
사전조건: 
- 믹서 패널 표시 상태
- 트랙 2개 이상이 있는 프로젝트
- 재생 중인 상태
절차:
1. 재생을 시작한다
2. 트랙 1의 M(뮤트) 버튼을 클릭한다
3. 트랙 1의 M 버튼을 다시 클릭하여 해제한다
기대결과:
- 뮤트 버튼 클릭 즉시 해당 트랙의 오디오가 중지된다
- 뮤트 해제 시 즉시 오디오가 재생된다
- 버튼이 활성 상태로 표시된다

### MX-011 트랙 솔로 (재생 중)
사전조건:
- 믹서 패널 표시 상태
- 트랙 2개 이상이 있는 프로젝트
- 재생 중인 상태
절차:
1. 재생을 시작한다
2. 트랙 1의 S(솔로) 버튼을 클릭한다
3. 트랙 1의 S 버튼을 다시 클릭하여 해제한다
기대결과:
- 솔로 버튼 클릭 즉시 다른 트랙의 오디오가 중지되고 트랙 1만 재생된다
- 솔로 해제 시 모든 트랙이 다시 재생된다
- 솔로 활성 트랙의 S 버튼이 활성 상태로 표시된다
- 다른 트랙의 M 버튼이 자동 뮤트 상태로 표시된다

### MX-012 뮤트/솔로 즉시 적용
사전조건:
- 믹서 패널 표시 상태
- 트랙 2개 이상이 있는 프로젝트
- 재생 중인 상태
절차:
1. 재생을 시작한다
2. 트랙 1의 M 버튼을 클릭한다
3. 트랙 2의 S 버튼을 클릭한다
4. 트랙 2의 S 버튼을 해제한다
5. 트랙 1의 M 버튼을 해제한다
기대결과:
- 각 버튼 클릭 즉시 오디오 출력이 변경된다
- 지연 없이 즉시 반영된다
- 버튼 상태가 정확하게 표시된다

### MX-013 솔로 트랙 뮤트 시 자동 뮤트 해제
사전조건:
- 믹서 패널 표시 상태
- 트랙 3개 이상이 있는 프로젝트
- 재생 중인 상태
절차:
1. 재생을 시작한다
2. 트랙 1의 S(솔로) 버튼을 클릭한다
3. 트랙 2와 트랙 3이 자동 뮤트 상태인지 확인한다
4. 트랙 1의 M(뮤트) 버튼을 클릭한다
5. 트랙 2와 트랙 3의 자동 뮤트 상태를 확인한다
기대결과:
- 트랙 1의 솔로가 활성화되면 트랙 2, 3이 자동 뮤트 상태가 된다
- 트랙 1의 뮤트를 누르면 트랙 1의 솔로가 해제되고 뮤트가 활성화된다
- 트랙 1이 솔로이면서 뮤트이면 사실상 아무 소리도 나지 않으므로, 트랙 2, 3의 자동 뮤트가 해제된다
- 트랙 2, 3의 M 버튼이 비활성 상태로 표시된다
